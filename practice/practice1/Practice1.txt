Hardload.

Agenda: Программа hardload.exe позволяет копировать, выбранные в списке файлы в указанный каталог. Программа написана специально для целей этого задания и далеко не идеальна.

Задание:

Выполнить исследовательское тестирование.

1.	Составить план тестирования программы hardload.exe (15 тестовых сценариев) = 10p.
2.	Внести тестовые сценарии в систему TestLink = 8p.
3.	Протестировать программу по плану, указав фактический результат для каждого тестового сценария. = 10p.
4.	Составить отчет о тестировании –  внести найденные дефекты в систему JIRA, в отчет добавить ссылки на созданные дефекты и краткое описание каждого дефекта. = 10p. 
5.	Написать пользовательскую документацию = 7p.
6.	Выслать план с результатами, отчет, документацию (Skype, VK). 

Пояснения:

Тестирование выполняется по методу "чёрного ящика". Код не тестируем.
Тестируем интерфейс (внешний вид) и функционал (работоспособность) программы.

Пользовательская документация.

1. Назначение ПО.
2. Условия использования ПО
3. Подготовка к работе
4. Работа с системой
