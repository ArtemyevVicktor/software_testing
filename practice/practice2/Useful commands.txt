Открытие страницы по заданному URL - open
Переход по ссылкам – click, clickAndWait 
Ввод значений - type 
Выбор значений из выпадающих списков – select

Verify*

verifyTextPresent
Использутсядля проверки наличия текста на странице.


verifyTitle
Позволяет проверить title страницы.

verifyElementPresent
Провряет наличие конкретного UI элемента на странице (кнопка, ссылка, поле ввода, итд)


Assert*

Команды assert работают аналогично командам verify. Единственное и важное отличие, если assert не выполняется, то прерывается весь тест и помечается как fail. Если не выполняется verify то тест продолжает работу, записывается ошибка в лог.

assertTextPresent
		Проверяет наличие текста

assertText
		Проверяет наличие текста у какого-либо UI элемента.

assertTitle
    		Проверяет корректность заголовка страницы

assertValue
Проверяет значение у поля ввода

assertElementPresent
		Проверяет наличие элемента на странице


waitFor*

waitForPageToLoad
Ждет полной загрузки страницы
Синтаксис waitForPageToLoad(timeout); timeout – максимальное время ожидания

waitForAlert
	Команда ждет появление всплывающего окошка с сообщением
waitForTable
	Ждет полной загрузки таблицы
waitForTitle
	Ждет появления страницы с нужным заголовком
waitForText
	Ждет появления текста на странице
