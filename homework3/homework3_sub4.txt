Analyses: По ссылке расположен bash-скрипт https://drive.google.com/file/d/0ByUW-8QzosXQOVBQSVA4a29nRTg/view?usp=sharing = 7p.
-----------------------------------------------------

Скрипт при запуске выдает сообщение "Please choose an option:"(Пожалуйста, укажите опцию)
1.Выполняет команду ls (содержимое каталога).
2.Выводит дату время в формате:
Time: %текущее время%
Date: %текущая дата% 
3.Выводит на экран сообщение: What file do you wish to check?(имя файла, который надо проверить). После ввода данных пользователем, выводит:
Если это директория: "%данные ввода% is a directory." %данные ввода%.
Если это файл: "%данные ввода% %s is a file." %данные ввода%.
Во всех остальных случаях: %данные ввода% does not exist. %данные ввода%.
4.Выводит на экран сообщение: "Please enter filename:"(Пожалуйста укажите имя файла). Копирует файл с расширением ".bak" в текущую директорию, если файл с таким именем существует.
5.Завершает работу скрипта.
Если вводимое пользователем значение, не число в интервале от 1 до 5, то показывает сообщение: "Not a valid option.".
